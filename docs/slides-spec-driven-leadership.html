How It Works

1. Specification First

Project constraints, tech stack, and quality standards defined in CLAUDE.md. AI reads this on every session.

2. AI-Assisted TDD

Claude Code writes tests first (RED), implements to pass (GREEN), refactors (REFACTOR). Human reviews and guides.

3. Automated Verification

Up to 17 quality gates run automatically based on work type: coverage, security, accessibility, E2E, mutation testing. No manual checks needed.

4. Provable Completion

Proof files document metrics: "90/90 tests passing, 93.37% coverage, 0 vulnerabilities." No more "I think it's done."
